# Lab 2: Primer Design

This GitHub section includes primer design based on example files from an RIT course and on the gene:

The gene sequence can be found here:  
[Microcystis aeruginosa FCY-28 McyE (mcyE) gene, complete cds](https://www.ncbi.nlm.nih.gov/nuccore/JQ290095.1)

Primer design was completed using command-line analyses, with code linked in this **Lab 2** folder.
